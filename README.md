# 🚀 Micro Lending Platform

A full-stack web application designed to manage micro-lending operations including customer management, loan processing, EMI tracking, and repayments.

---

## 📌 Features

* 🔐 JWT-based Authentication & Authorization
* 👥 Customer Management
* 💰 Loan Application & Approval System
* 📊 EMI Schedule Generation
* 💳 Repayment Tracking
* ⚠️ Penalty Calculation
* 📈 Admin Dashboard

---

## 🛠️ Tech Stack

### 🔹 Backend

* Java
* Spring Boot
* Spring Security (JWT Authentication)
* Hibernate / JPA
* PostgreSQL

### 🔹 Frontend

* React (Vite)
* JavaScript
* CSS

---

## 📁 Project Structure

```
micro-lending-platform/

```

---

## ⚙️ Setup Instructions

### 🔧 Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Backend will run at:

```
http://localhost:8081
```

---

### 💻 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run at:

```
http://localhost:5173
```

---

## 🔗 API Integration

Frontend communicates with backend using REST APIs:

```
http://localhost:8081/api/
```
